## My Color Wheel

* Tool for generating custom color wheels.
* Run the tool [here](https://xankill3r.github.io/my-color-wheel/).
* Inspired by Color and Light by James Gurney

- Uses the [Chrome.js](https://github.com/gka/chroma.js) library.
